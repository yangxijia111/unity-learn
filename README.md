## 🎯 学习路线

1. **第一阶段**（1-2周）：C# 语法基础 → 能写简单控制台程序
2. **第二阶段**（2-3周）：Unity 基础操作 → 能做简单 2D 游戏
3. **第三阶段**（3-4周）：Unity 核心系统 → 掌握常用功能
4. **第四阶段**（4-6周）：进阶技术 → 能独立开发中等复杂度游戏
5. **第五阶段**（持续）：实战项目 → 积累作品集

## 💡 使用建议

- 每个文件夹按编号顺序学习
- 每个主题都有示例代码和注释
- 建议边看边动手实践
- 遇到问题先尝试自己解决，再查阅资料
