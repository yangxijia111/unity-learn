## Unity Hub 简介

Unity Hub 是管理 Unity 项目的入口工具：

- **安装管理**：管理多个 Unity 版本（LTS / Beta / Alpha）
- **项目管理**：创建、打开、固定常用项目
- **模板选择**：2D、3D、URP、HDRP 等项目模板
- **许可证激活**：管理个人版/专业版许可证

## 编辑器界面概览

Unity 编辑器由多个窗口面板组成，可以自由排列：

| 面板 | 作用 |
|------|------|
| **Scene** | 场景视图，编辑和摆放 GameObject |
| **Game** | 游戏视图，预览最终运行效果 |
| **Hierarchy** | 层级面板，显示场景中所有对象的树状结构 |
| **Project** | 项目面板，管理所有资源文件（脚本、贴图、预制体等） |
| **Inspector** | 检查器，查看和编辑选中对象的组件属性 |
| **Console** | 控制台，查看 Debug 输出、警告和错误 |

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Q` | 手平移工具（平移视图） |
| `W` | 移动工具（Move） |
| `E` | 旋转工具（Rotate） |
| `R` | 缩放工具（Scale） |
| `T` | 矩形工具（Rect） |
| `F` | 聚焦选中物体 |
| `Ctrl+D` | 复制选中对象 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+S` | 保存场景 |
| `Ctrl+P` | 播放/暂停游戏 |
| `Space` | 切换工具模式 |

## 项目创建与管理

```csharp
// 项目创建在 Unity Hub 中完成：
// 1. 打开 Unity Hub → "New Project"
// 2. 选择模板（2D/3D/URP）
// 3. 选择保存路径和项目名称
// 4. 点击 "Create project"

// 项目结构（自动生成）：
// Assets/        → 你的资源（脚本、贴图、预制体等）
// Packages/      → 包管理（Package Manager 的依赖）
// ProjectSettings/ → 项目设置
// Library/       → 缓存（不要手动修改）
```

## 基本工作流

典型的游戏开发工作流：

```
1. 创建场景（Scene）
      ↓
2. 搭建场景（拖入 GameObject、地形、灯光）
      ↓
3. 编写脚本（C# 脚本控制行为）
      ↓
4. 测试运行（Play 模式）
      ↓
5. 调试修复（Console 查看日志）
      ↓
6. 打包发布（Build Settings）
```

### 工作流小贴士

- **频繁保存**：`Ctrl+S` 是好习惯
- **善用 Prefab**：重复使用的对象做成预制体
- **版本控制**：项目大了之后考虑用 Git 管理
- **Console 面板**：调试时永远第一个看的地方
